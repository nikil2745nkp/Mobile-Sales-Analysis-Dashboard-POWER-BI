### 📱 Mobile Sales Dashboard | Power BI
## Overview

This project presents an interactive Mobile Sales Dashboard developed using Power BI to analyze mobile phone sales performance across different cities, brands, payment methods, and time periods.

The dashboard provides actionable insights into sales trends, customer purchasing behavior, transaction patterns, and brand performance through dynamic visualizations and filters.

## Dashboard Features
KPI Cards
Total Sales: 769M
Total Quantity Sold: 19K
Total Transactions: 4K
Average Sales Value: 40K

## Key Insights
Apple generated the highest overall sales revenue.
iPhone SE emerged as one of the top-selling mobile models.
UPI and Card payments contributed significantly to total transactions.
Major metropolitan cities showed the highest sales concentration.
Monthly sales trends help identify peak demand periods.
